# Your Energy — Fitness & Workout Application

This is my final coursework for the Javascript course on GoIT. A recreation of a fitness/gym web-mockup made in Figma.

## Technology
* **HTML5 / CSS3**
* **JavaScript (ES6+)**
* **Vite**
* **REST API** — interaction with the backend (quotes, exercises, subscription)
* **LocalStorage** — saving selected exercises and caching quote of the day
